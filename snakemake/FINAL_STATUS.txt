🎯 WHAT EACH MODULE DOES
═════════════════════════════════════════════════════════════════════════════

QC & QUALITY CONTROL:
  fastqc.smk
    Input:  FASTQ files
    Output: HTML quality reports
    Use:    Check raw read quality

  lib_complex_preseq.smk
    Input:  BAM file
    Output: Learning curve extrapolation
    Use:    Estimate library complexity

  frag_length_distribution.smk
    Input:  BAM file
    Output: Fragment length distribution
    Use:    Analyze fragment sizes

  filter_properly_paired.smk
    Input:  BAM file
    Output: Filtered BAM (paired reads only)
    Use:    QC filtering for paired-end data

  quality_reports.smk
    Input:  All QC files
    Output: CSV summary + HTML report
    Use:    Aggregate QC metrics

ANALYSIS & ANNOTATION:
  peaks_annotation.smk
    Input:  narrowPeak + GTF
    Output: Annotated peak file
    Use:    Add genomic feature information

  motif_gc_content.smk
    Input:  BAM + genome FASTA
    Output: Motif BED file
    Use:    Analyze nucleotide patterns

  enrichment_analysis.smk
    Input:  BED + on/off-target files
    Output: Enrichment ratios
    Use:    Quantify enrichment

  snp_fingerprint.smk
    Input:  BAM files + SNP reference
    Output: Fingerprint report
    Use:    Verify sample identity
